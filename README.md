 ## Features

- *User Authentication: Secure login for admins, project managers, and team members.
- *Project Management: Create, update, and monitor projects with timelines.
- *Task Assignment & Tracking: Assign tasks and track real-time progress.
- *Dashboards for Each User Role: Dedicated views for admin, project manager, and team member.
- *Progress Reports & Visualizations: View charts for performance and task completion.
- *Profile Management: Users can update account information.
- *System Configuration: Admins can manage platform-wide settings.

### User Types & Capabilities

1. Admin
-Manages users, projects, and system settings.
-Functionalities:
-user Management:
-Input: name, email, role
-Output: success notification for creation/update/deletion


2. Project Manager

-Creates and manages projects, assigns tasks, and monitors progress.
-Functionalities:
-Project Creation:
-Input: project details
-Output: project creation confirmation
-Task Assignment:
-Input: task details + assignee
-Output: task assigned message


3. Team Member

-Tracks tasks, updates status, and views project information.
-Functionalities:
-Task Tracking:
-Input: task updates
-Output: updated task status
