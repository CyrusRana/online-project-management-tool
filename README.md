üë§ User Types & Capabilities
1. Admin

Manages users, projects, and system settings.

Functionalities:

User Management:

Input: name, email, role

Output: success notification for creation/update/deletion

Project Management:

Input: title, description, timeline

Output: confirmation on creation/update/deletion

System Settings:

Input: configuration options

Output: settings updated message

2. Project Manager

Creates and manages projects, assigns tasks, and monitors progress.

Functionalities:

Project Creation:

Input: project details

Output: project creation confirmation

Task Assignment:

Input: task details + assignee

Output: task assigned message

Progress Monitoring:

Input: project progress data

Output: progress charts & insights

3. Team Member

Tracks tasks, updates status, and views project information.

Functionalities:

Task Tracking:

Input: task updates

Output: updated task status

Project Details:

Input: project selection

Output: timeline & project overview

Profile Management:

Input: name, email, password

Output: profile updated notification
